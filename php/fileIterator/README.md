# File Iterator

Дан текстовый файл размером 2ГБ. Напишите класс, реализующий интерфейс SeekableIterator, для чтения данного файла.

# Solve

Implement ```\SeekableIterator```. Use fseek
See ```read.php``` for more details

#How to use

1 . Install dependencies

```
composer install
```

2 . Generate file

```
php generate.php
```

3 . Read File

```
php read.php
```
